rangesldier.js author and link providied in html comments.

Music clips are from Code Monkey by Jonathan Coulton, go listen to some of his music on YouTube, you're welcome!

    How to use this Pomodoro
  • Use sliders to adjust Work and Break Times, and Intervals
  • Work Time can be set for up to 50 minutes
  • Break Time can be set for up to 25 minutes
  • Intervals can be set up to 8

  • Press the Play button to start timer
    • Play will be disabled until reset or ALL intervals have been completed
  • The Pause button will pause and restart the timer from a paused state
  • Press the Reset button to return to default timer settings
    Features or Bugs depending on your perspective...
  • You CAN set Work or Break to zero and the timer will still count down
  • Interval can be zero and you will get one pass through either Work or Break, but Break will not fire if Interval is zero and Work has a value
  • Flipping to this screen is quirky and probably won't work in all browsers. I didn't write flip, and I didn't quite grasp how to get jquery to work with a much better vanilla CSS card flipper. I think if I could have used vanilla js for this project it wouldn't have been an issue. Since I didn't want to roll my own slider code I just tried to use jquery throughout this project.
